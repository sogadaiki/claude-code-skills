# 法務案件用テンプレート（legal / documents）

**注意事項**（テンプレート適用前に必ず確認）:
- 個人情報（住所、電話番号、メールアドレス、口座番号）は一切記載しない
- 記載する当事者情報は氏名・肩書・役職のみ
