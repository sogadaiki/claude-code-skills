# 週次ビジネスレビュー

以下のサブエージェントを順次起動し、多角的な週次レビューを実施します。

## 1. 事業進捗チェック（mei-business-strategist）
- 今週のOKR進捗
- 重要な意思決定事項
- ブロッカーの有無

## 2. 財務チェック（rina-finance-advisor）
- 今週の売上/支出
- キャッシュフロー状況
- 予算対実績の乖離

## 3. マーケティングチェック（kenji-marketing-strategist）
- SNSメトリクス（フォロワー、エンゲージメント）
- リード獲得数
- コンテンツパフォーマンス

## 4. 開発チェック（GitHub CLI直接取得）
- 各リポジトリのopen Issues / 最近マージされたPRを `gh` コマンドで取得
- 開発マイルストーン進捗
- 技術的負債の状況
- リリース予定

## レビュー手順

1. まずユーザーに今週のハイライト/ローライトを聞く
2. 各領域の定量データを確認
3. 課題と対策を整理
4. 来週のTop 3アクションを決定
5. 結果をメモリに保存（前週比較のため）
